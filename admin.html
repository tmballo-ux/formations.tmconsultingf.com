🚀 Processus de mise à jour

        

        

          

            
Comment modifier et déployer

            
    
              
  • 
                
    1
    
                
    Utilise l'Assistant IA du Dashboard pour décrire ta modification en français et obtenir le code HTML corrigé.
    
              
    • 
              
  • 
                
    2
    
                
    Va sur GitHub → formations.tmconsultingf.com et ouvre le fichier à modifier.
    
              
    • 
              
  • 
                
    3
    
                
    Clique le crayon ✏️ pour éditer, colle le code corrigé, puis clique "Commit changes".
    
              
    • 
              
  • 
                
    4
    
                
    Netlify détecte le commit et redéploie automatiquement. Ton site est mis à jour en 30 à 60 secondes.
